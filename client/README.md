# USOF Application

This is the Client side for the USOF (User Stack Overflow Forum) application. Follow the instructions below to set up and run the project.

## Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/lerobait/UCODE-Usof.git
   ```

2. **Navigate to the client directory:**:

   ```bash
   cd client
   ```

3. **Install dependencies**:

   To install dependencies run:

   ```bash
   npm install
   ```

4. **Start the client:**:

   Finally, start the project:

   ```bash
   npm run start
   ```
